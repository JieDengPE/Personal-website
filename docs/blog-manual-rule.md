# Blog vs Manual Rule

如果你不确定一份内容该放 `Blog` 还是 `Manuals`，用下面这个判断：

## Put It In Blog

如果这篇内容主要是：
- 研究思考
- 技术判断
- 阅读笔记
- 阶段总结
- 某个问题为什么重要

那么它应该进 `Blog`。

## Put It In Manuals

如果这篇内容主要是：
- 操作步骤
- 实验流程
- 上电 checklist
- 调试顺序
- 平台搭建说明
- 可重复执行的方法

那么它应该进 `Manuals`。

## Fast Rule

- `Blog` = insight
- `Manuals` = procedure

## Good Examples For This Site

适合 Blog：
- 为什么中压 SST 值得做
- 从三电平逆变器到中压系统研究的转变
- AI 在功率模块设计里真正有用吗

适合 Manuals：
- 局部放电平台上电流程
- GaN 模块开关测试步骤
- 热测试平台的准备与检查顺序
